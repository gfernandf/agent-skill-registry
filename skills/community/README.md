# Community Skills

Skills in this channel are **contributor-owned workflows** that have passed
peer review and the full admission checklist, but have not yet been promoted
to the official catalog.

## Who can contribute

Any contributor via PR.

## Requirements

All six items in the `docs/SKILL_ADMISSION_POLICY.md` admission checklist
must be answered in the PR description.  At least one peer review is required
before merge.

## Stability guarantee

Interfaces are expected to be stable within minor versions.  Breaking changes
require a semver major bump and a migration note.

## Next step

Maintainers promote community skills to `official/` after they demonstrate
sufficient usage and quality evidence (smoke tests, lab verification score).
`official/` is maintainer-led curation, not the default first publication step.
See `docs/SKILL_ADMISSION_POLICY.md` for the full channel model.
